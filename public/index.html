브라우저 문제 해결

웹페이지가 수정됐는데 예전 화면이 계속 보일 때 해결 방법

캐시 때문에 새 내용이 보이지 않을 때 강력 새로고침, 시크릿 창, 배포 상태 확인을 순서대로 점검합니다.

계정 보안

구글 계정을 안전하게 쓰기 위한 기본 보안 점검

복구 이메일, 2단계 인증, 로그인 기기 확인처럼 계정을 잃어버리지 않기 위해 먼저 확인할 항목을 설명합니다.

파일 관리

Windows에서 파일 확장자를 표시해야 하는 이유와 설정 방법

파일 이름 뒤의 .pdf, .jpg, .exe를 보이게 설정해 잘못된 파일 실행을 줄이는 방법을 안내합니다.
