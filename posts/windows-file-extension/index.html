파일 관리

Windows에서 파일 확장자를 표시해야 하는 이유와 설정 방법

작성일: 2026년 7월 12일 · 예상 읽기 시간: 4분

파일 이름 뒤의 .pdf, .jpg, .exe 같은 부분을 확장자라고 합니다. 확장자를 보이게 해두면 파일 종류를 더 정확하게 확인할 수 있습니다.

1. 확장자가 중요한 이유

Windows는 기본 설정에서 알려진 파일 형식의 확장자를 숨길 수 있습니다. 이 경우 안내문.pdf처럼 보이는 파일이 실제로는 안내문.pdf.exe일 수도 있습니다. 확장자를 숨기면 사용자는 마지막 .exe를 보지 못하고 PDF라고 착각할 수 있습니다.

모든 파일이 위험한 것은 아니지만, 출처가 불분명한 파일을 열 때 확장자를 확인하는 습관은 안전에 도움이 됩니다.

2. Windows 11에서 확장자 표시하기

  1. 파일 탐색기를 엽니다.
  2. 상단 메뉴에서 보기를 선택합니다.
  3. 표시 메뉴로 들어갑니다.
  4. 파일 확장명 항목을 켭니다.

3. Windows 10에서 확장자 표시하기

  1. 파일 탐색기를 엽니다.
  2. 상단의 보기 탭을 선택합니다.
  3. 파일 확장명 체크박스를 켭니다.

4. 자주 보는 확장자

  • .pdf: 문서 파일
  • .jpg, .png: 이미지 파일
  • .docx, .xlsx: Word, Excel 문서
  • .zip: 압축 파일
  • .exe, .msi: 실행 또는 설치 파일

5. 파일 이름을 바꿀 때 주의할 점

확장자를 표시한 뒤 파일 이름을 바꿀 때는 마지막 확장자를 실수로 지우지 않도록 주의합니다. 예를 들어 사진.jpg에서 .jpg를 지우면 Windows가 어떤 프로그램으로 열어야 할지 헷갈릴 수 있습니다.

정리

확장자를 표시하는 것은 작은 설정이지만 파일을 안전하게 확인하는 데 도움이 됩니다. 특히 이메일 첨부파일이나 메신저로 받은 파일을 열기 전에는 파일 이름 전체를 확인하는 습관을 들이는 것이 좋습니다.
